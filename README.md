# MY FEM project #

This project is being implemented as part of the course 'Non-linear finite elements'.

### Guidelines ###

* This project aim to be an educational framework for solving partial differential equations using the finite element method.
* Version 0.1

### How do I get set up? ###

	git clone https://bitbucket.org/hernanmella/myfem
	cd myfem && sudo python setup.py install

### Dependencies ###
* [meshio](https://github.com/nschloe/meshio)
* [SciPy](https://www.scipy.org/)
* [NumPy](http://www.numpy.org/)
* [PETSc4Py](https://bitbucket.org/petsc/petsc4py)
